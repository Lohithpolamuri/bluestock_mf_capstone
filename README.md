Mutual Fund Analytics Capstone Project

Project Overview

This project analyzes mutual fund industry data to generate insights into fund performance, investor demographics, SIP trends, category inflows, and market behavior. The objective is to help investors understand industry trends and make informed investment decisions through data analytics and interactive dashboards.

Tools & Technologies

- Python
- Pandas
- NumPy
- SQL
- Tableau Public
- Git
- GitHub

Project Structure

- data/ : Raw and processed datasets
- notebooks/ : Analysis notebooks
- scripts/ : ETL and analytics scripts
- sql/ : SQL queries and schema files
- dashboards/ : Dashboard files and screenshots
- reports/ : Final report and presentation

Datasets Used

The project uses mutual fund datasets containing information on:

- Assets Under Management (AUM)
- SIP Inflows
- NAV History
- Fund Categories
- Investor Demographics
- Portfolio Holdings
- Scheme Performance

Dashboard

Tableau dashboards were created to analyze:

- Industry Overview
- Fund Performance
- Investor Demographics
- SIP & Market Trends

How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the analytics scripts from the scripts folder.

3. Open the Tableau dashboard using the provided Tableau Public link.
